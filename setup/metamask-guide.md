# MetaMask setup

1. Install MetaMask.
2. Create or import a wallet.
3. Save your seed phrase offline.
4. Switch to Sepolia for Track 1, 2, and 4.
5. Switch to Avalanche Fuji for the CCIP sender side in Track 3.

## Why this matters

Your wallet is how you sign deployment and interaction transactions. For the workshop, the wallet is the student’s control panel.

## Common beginner mistakes

- Using mainnet instead of testnet
- Copying the wrong address
- Forgetting to select the correct network before sending a transaction
