# Chainlink Builder Lab Vadodara - Workshop Day Guide



## Before You Arrive



Please install:



* Node.js 22 LTS or newer

* Git

* MetaMask Browser Extension

* VS Code



## Required Accounts



Create:



* GitHub Account

* Alchemy Account

* MetaMask Wallet



## Workshop Setup



Clone the repository:



```bash

git clone https://github.com/Saurabhnage/chainlink-builder-lab-vadodara.git



cd chainlink-builder-lab-vadodara

```



Install dependencies:



```bash

npm install

```



Create environment file:



```bash

copy .env.example .env

```



Verify setup:



```bash

npm run workshop:check

```
